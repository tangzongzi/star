# 儿童奖励系统项目配置 (MVP版本)

## 项目概述
"星星任务"是一个面向家庭的儿童奖励系统，旨在通过任务完成和积分奖励的方式，鼓励孩子养成良好习惯。家长可以创建任务，设置奖励，而孩子则通过完成任务获取积分，兑换各种奖励。

## 技术栈
- **前端**: Vue3 + Vite
- **样式**: TailwindCSS
- **3D效果**: Three.js
- **部署**: Netlify
- **数据存储**: 本地存储 (LocalStorage)，后续可扩展到云存储

## 关键模式和约定
- **代码风格**: 使用ES6+语法，组合式API (Composition API)
- **文件结构**: 基于特性的模块化组织
- **状态管理**: Pinia或Vue3 Reactive状态
- **命名约定**: camelCase (变量和函数)，PascalCase (组件)

## 功能需求

### 1. 简化用户管理 [MVP功能]
- **模式切换**: 简单的家长/孩子模式切换，无需复杂认证
- **单一家庭**: 仅支持一个家庭的数据管理

### 2. 任务管理 [MVP功能]
- **任务创建**: 家长创建任务，设置名称、描述、点数
- **任务类型**:
  - 一次性任务（如完成特定作业）
- **任务分类**: 简单分类（学习、家务、其他）
- **任务状态**: 未完成、等待验证、已完成
- **任务验证**: 孩子标记完成，家长确认

### 3. 积分系统 [MVP功能]
- **积分获取**: 完成任务获得积分
- **积分余额**: 显示当前可用积分

### 4. 奖励系统 [MVP功能]
- **奖励商店**: 家长创建可兑换的奖励
- **奖励类型**:
  - 物质奖励（如玩具、书籍）
  - 特权奖励（如额外的游戏时间、选择晚餐）
- **奖励兑换**: 使用积分兑换奖励

### 5. 3D展示功能 [MVP功能]
- **成就墙**: 简单的3D展示墙，展示已获得的奖励
- **基础动画**: 完成任务或兑换奖励时的简单庆祝动画

### 6. 数据存储 [MVP功能]
- **本地存储**: 使用localStorage存储所有数据
- **导出/导入**: 简单的数据备份和还原功能

## 数据模型

### Task (任务)
```javascript
{
  id: "任务ID",
  title: "任务标题",
  description: "任务描述",
  points: "积分奖励",
  category: "任务分类",
  status: "任务状态", // 未完成、等待验证、已完成
  createdAt: "创建时间",
  completedAt: "完成时间"
}
```

### Reward (奖励)
```javascript
{
  id: "奖励ID",
  title: "奖励名称",
  description: "奖励描述",
  points: "所需积分",
  category: "奖励类型",
  imageUrl: "奖励图片(可选)",
  createdAt: "创建时间",
  redeemed: "是否已兑换"
}
```

### PointTransaction (积分交易)
```javascript
{
  id: "交易ID",
  points: "积分数量",
  type: "获取/使用",
  source: "任务ID/奖励ID",
  description: "交易描述",
  createdAt: "创建时间"
}
```

## 页面结构

### 主界面
- **模式选择**: 家长/孩子模式切换

### 家长模式
- **任务管理**: 创建、编辑、删除任务，验证任务完成
- **奖励管理**: 创建、编辑、删除奖励，确认奖励兑换
- **统计概览**: 简单的任务完成和奖励兑换统计

### 孩子模式
- **任务列表**: 查看分配的任务，标记完成
- **奖励商店**: 查看可用奖励，使用积分兑换
- **成就墙**: 3D展示已获得的奖励和成就

## UI/UX设计标准 [简化版]

### 响应式设计
- 移动优先设计
- 支持平板和桌面端的扩展布局

### 交互元素
- 简洁直观的按钮和表单
- 任务和奖励使用卡片式布局
- Three.js场景集成到主界面
